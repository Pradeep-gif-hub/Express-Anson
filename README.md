## ğŸ§© What is a Framework?

A **framework** is a **predefined structure** that helps developers build applications efficiently by providing:
- Ready-made tools
- Standard architecture
- Reusable components
- Best practices

Instead of writing everything from scratch, a framework lets you **focus on application logic** while it handles common tasks like routing, request handling, and middleware execution.

### Backend Frameworks Do:
- Handle HTTP requests & responses
- Manage routing
- Support middleware
- Connect databases
- Improve scalability and maintainability
